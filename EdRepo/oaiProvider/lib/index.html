OAI-PMH Provider for SWEnet

The OAI-PMH Provider for SWEnet serves metadata contained in the SWEnet database to OAI-PMH harvesters. The provider is written in PHP from scratch by Ethan Greer at Drexel University. Inquiries about the provider should be directed to:

elg42 at drexel dot edu

Use the Provider

Point your OAI-PMH version 2 compliant harvester to:

http://www.xcitegroup.org/oaiProvider/provider.php

to harvest requests from the provider.

You can also see some basic information about the provider from your browser at:

http://xcitegroup.org/oaiProvider/provider.php?verb=Identify

and looking at the page source.

Notes

  • This provider doesn't support sets or resumption tokens. This isn't much of an issue currently, given SWEnet's small size.
  • This provider passes the OAI Repository Explorer's automated compliance tests. It is compliant with the OAI-PMH protocal version 2 at least to the extent tested by the Repository Explorer.
